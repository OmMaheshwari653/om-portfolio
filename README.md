## Getting Started

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Run linting

```bash
npm run lint
```

## Project Structure

- src/App.jsx handles the page layout and section order
- src/sections/ contains the main portfolio sections
- src/components/ contains reusable UI pieces like buttons and cards
- src/index.css defines the global styling and animations
